Multiple Multipliers

This game was made by cionx over one week for the Minimalist Game Jam #1. The theme of the Jam was “Multi”.
The game is written in HTML, CSS and TypeScript. The code is publicly available at https://gitlab.com/cionx/multiple-multipliers.
The font used is Valera Round, which is published under OLF 1.1 on Google Fonts.
